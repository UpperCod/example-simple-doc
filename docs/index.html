Welcome a simple docs

Create quick singlePage documentation with Markdown + Webcomponents

Atomico is a library for the creation of interfaces only using functions.

Introduction

Create documentation in a simple way, this configuration allows you to extract all the html and markdow files from your src/ directory and group them into a single document, but with a big difference use webcomponetns freely, the configuration will dynamically resolve all Assets exports and groups them using Rollup. It is the magic of EStack

Configuration

By default the template will take the configuration declared in the index.md file, eg:

title: Welcome a simple doc
description: Create quick singlePage documentation with markdown + webcomponents
hero: false # enable the header showing title and description
order: 0 # Define the order of the section
columns: true # Activate the use of columns in the section
socials:
  twitter: https://twitter.com/uppercod
  github: https://github.com/
files:
  logo: ./logo-white.svg
fetch:
  author: https://api.github.com/users/atomicojs # export the author from github

The minimum metadata to create a new section within documentation is:

---
order: 1
---

Template configuration

The template is inside the src/template, it has local assets that can be modified to change all its behavior.

input-password

input-password, input that allows to show or hide the password based on a state such as owner or click event

Installation

npm install atomico-base

Usage

import "atomico-base/input-password"
